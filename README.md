## 🌟 プロジェクト概要

**림딩동**は、学校のサークルを効果的に広報し、学生にサークル情報を提供するために開発されたWebプラットフォームです。

## ✨ 主要機能

### 1. サークル紹介
- 分野別サークルのスライド表示
- クリックで詳細なサークル情報を提供

### 2. サークル占いページ
- カスタマイズされたアンケートによるサークル占いシステム

### 3. サークル部室の場所案内ゲーム
- WebGLベースのインタラクティブなキャンパスマップ
- サークル部室の位置と情報を提供

### 4. メインページ
- 多彩なアニメーション演出
- スクロールイベントとインタラクティブなデザイン
